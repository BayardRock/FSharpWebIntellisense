# About FSharpWeb
FSharpWeb is an example of how to implement intellisense for F# over [SignalR](http://signalr.net/) or [WebAPI](http://www.asp.net/web-api)
using the [ACE editor](http://ace.c9.io/).

# Demos
Download solution and run for demos.
